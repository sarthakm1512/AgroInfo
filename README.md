## ğŸ”„ Workflow and Working

The application starts from index.html, which acts as the main entry point. Styling and layout are handled by index.css, ensuring responsiveness and consistency across devices. JavaScript logic in index.js manages interactivity and dynamic behavior such as navigation handling and DOM updates. Additional informational sections are organized into separate HTML files within the pages directory. Static data and image assets are loaded locally to ensure performance, reliability, and offline usability. The user can navigate through the platform seamlessly without authentication or backend dependencies.
